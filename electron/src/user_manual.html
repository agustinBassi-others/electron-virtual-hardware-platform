Manual de Uso
Virtual Hardware Platform

 

Author: Agustin Bassi

 

Descripción conceptual

 

El propósito de la aplicación es ser una herramienta que permita la enseñanza de programación en Sistemas Embebidos mediante una interfaz amigable y fácil de utilizar.

 

A grandes rasgos, el proyecto se compone de dos partes fundamentales:

  • Una aplicación de escritorio multiplataforma creada en Electron (NodeJS, HTML y CSS) que contiene periféricos virtuales.
  • Una biblioteca para Sistemas Embebidos creada en Lenguaje C para múltiples arquitecturas que permite acceder a los periféricos virtuales de la aplicación mediante el puerto Serie-USB.

 

Instalación

 

Descargar de este link el binario para el sitema operativo deseado (Linux o Windows) y seguir los pasos de instalación convencionales.

 

Una vez instalado abrir el programa y explorar las diferentes solapas.

 

Conexión con el sistema embebido

 

  1. Conectar el sistema embebido
  2. Dirigirse a la solapa periféricos.
  3. Elegir el puerto donde se encuentra conectado el embebido y la velocidad de conexión.
  4. Clickear sobre la llave de conexión.
    1. Si cambió a color verde, la conexión está establecida.
    2. Si quedó en color rojo chequear que el puerto seleccionado sea el correcto.
    3. En caso que de haber conectado el sistema embebido después de iniciar el programa, presionar CTRL + R para recargar el programa.

 

Hola Mundo con Plataforma de Hardware Virtual

 

  • Sistema operativo Linux o Windows.
  • Sistema embebido con puerto USB-Serie.
  • IDE para compilar programa (ciaa-ide, embeddedIde, Arduino Sketch, etc.).
