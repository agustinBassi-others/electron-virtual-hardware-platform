Instrucciones

1- Abrir IDE para descargar firmware y crear proyecto
2- Crear tres archivos con los siguientes nombres:
3- main.c - vihard.h - vihard.c
4- Dirigirse a la solapa “Firmware Embebido” de ViHard
5- Copiar los archivos correspondientes
6- Seleccionar la BOARD que se va a programar en vihard.h
7- Compilar el programa y descargarlo al sistema embebido
8- Conectar el sistema embebido a este programa
9- FIN!!! Ver el comportamiento del hardware virtual
