Instrucciones para programar
el sistema embebido

  1. Abrir software para programar el sistema embebido y crear nuevo proyecto
  2. Crear tres archivos con los siguientes nombres:
    vihard.h - vihard.c - test_vihard.c
  3. Copiar el contenido de los archivos de esta pestaña en los archivos creados previamente
  4. Seleccionar la 'BOARD' a programar editando las línes 41-43 del archivo vihard.h
  5. Compilar el programa y descargarlo al sistema embebido
  6. Conectar el sistema embebido a este programa por USB
  7. Obsevar el comportamiento del hardware virtual
