Instrucciones para programar
el sistema embebido

  1. Abrir el software para programar el sistema embebido y crear un nuevo proyecto
  2. Dentro del nuevo proyecto crear tres archivos:
    vihard.h - vihard.c - vihard_examples.c
  3. Copiar el contenido de cada archivo de esta pestaña en los archivos creados dentro del proyecto
  4. Seleccionar la 'BOARD' a programar editando las líneas 41-44 del archivo vihard.h (por defecto EDU-CIAA-NXP)
  5. Compilar el programa y descargarlo al sistema embebido
  6. Conectar el sistema embebido a este programa por USB
  7. Presionar VH_TEC4 para ver los diferentes ejemplos
